# Soter AI Security Guard — Screenshots Checklist

## Required Screenshots for Store Listing

### Chrome Web Store
- **Size:** 1280×800 or 640×400 pixels
- **Format:** PNG or JPEG
- **Minimum:** 1 screenshot
- **Maximum:** 5 screenshots

### Microsoft Edge Add-ons
- **Size:** 1280×800 pixels (16:10 ratio)
- **Format:** PNG or JPEG
- **Minimum:** 1 screenshot
- **Maximum:** 8 screenshots

## Recommended Screenshots

### Screenshot 1: Extension Popup — Enrolled State
- Show the extension popup with green "Enrolled" status
- Display employee name and department
- Show the "Scan active" indicator
- Caption: "Enterprise enrollment with active AI security monitoring"

### Screenshot 2: Prompt Blocked Alert
- Show a ChatGPT page with a Soter AI overlay blocking a prompt
- The blocked prompt should show redacted content (e.g., `[REDACTED_API_KEY]`)
- Display the block reason: "Secret detected in prompt"
- Caption: "Real-time blocking of sensitive data in AI prompts"

### Screenshot 3: Side Panel — Policy Status
- Show the side panel with current policy rules
- Display the last sync time and policy version
- Show active destination list
- Caption: "View active security policy and sync status"

### Screenshot 4: Emergency Lockdown (Admin View)
- Show the admin dashboard with emergency lockdown active
- Display the red "LOCKDOWN ACTIVE" banner
- Show the list of restrictions
- Caption: "One-click emergency lockdown across all enrolled devices"

### Screenshot 5: Shadow AI Discovery (Admin View)
- Show the admin dashboard with discovered AI tools
- Display the unknown destinations list
- Show approve/block/classify action buttons
- Caption: "Discover and manage shadow AI usage across the organization"

## Screenshot Guidelines
- Use realistic but non-sensitive data in all screenshots
- Blur or redact any real API keys, passwords, or PII
- Show the extension in both light and dark themes if possible
- Include captions that explain what the screenshot demonstrates
- Do NOT show raw secrets, real employee data, or production URLs
