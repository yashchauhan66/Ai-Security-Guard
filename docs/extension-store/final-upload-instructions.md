# Final Upload Instructions

Use only after signing into the relevant publisher account.

## Artifact
- Extension ZIP: `apps/extension/dist/soter-extension-v0.1.0.zip`

## Chrome Web Store
1. Open Chrome Web Store Developer Dashboard.
2. Create or select the Soter Enterprise AI Guard item.
3. Upload the ZIP above.
4. Fill listing from `docs/extension-store/chrome-web-store-listing.md` or private listing from `docs/extension-store/chrome-private-listing.md`.
5. Upload screenshots from `docs/extension-store/screenshots`.
6. Upload promo image `docs/extension-store/screenshots/promo-large-920x680.png`.
7. Fill privacy practices from `docs/extension-store/privacy-policy.md` and permission notes from `docs/extension-store/permission-justification.md`.
8. Add reviewer notes from `docs/extension-store/reviewer-notes.md`.
9. Submit for review.

## Edge Add-ons
1. Open Microsoft Partner Center / Edge Add-ons.
2. Upload the same ZIP.
3. Fill listing from `docs/extension-store/edge-addons-listing.md` or hidden listing from `docs/extension-store/edge-hidden-listing.md`.
4. Upload screenshots and privacy/reviewer docs.
5. Submit for certification.
