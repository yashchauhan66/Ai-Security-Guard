# Customer Limitations

Date: 2026-07-02

- Browser extension is beta `0.1.0`; external AI site DOM changes can temporarily reduce interception quality.
- PDF/DOCX/XLSX/PPTX handling is metadata-only in the extension package; do not claim deep parsing for these formats.
- Semantic fingerprinting must not be claimed as complete beyond implemented tests and current product behavior.
- Public marketplace availability depends on Chrome, Edge, npm, n8n, Zapier, and Make account/review gates.
- Authenticated enterprise enrollment is required for full extension behavior.
